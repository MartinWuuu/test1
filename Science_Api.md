
## 1，登录

### 参数

- 用户名
- 密码

### 返回

- status (0或1，请求错误返回 0 ，请求成功返回 1)
- info （当请求失败时提示的信息）
- data 用户id

## 3，获取各种列表

### 参数

- 标识码 【必填】（0代表获取课件，1代表获取教案，2代表获取课程资源，3代表获取名师讲堂，4代表获取新闻公告）
- 年级【选填】（3，4，5，6）
- 学期 【选填】（上学期，下学期）

### 返回

- status (0或1，请求错误返回 0 ，请求成功返回 1)
- info （当请求失败时提示的信息）
- data array【名称，url】
